Vibranium

A comprehensive study of managing data constraints in database-backed web applications

What is Vibranium?

Vibranium is the the first in-depth study of data-constraint problems in web applications. Our study provides moti-vations and guidelines for future research to help developers better manage data constraints.

You can find the source code of Vibranium in our github repository.
